Use a Template in a Signature Request

This recipe uses an existing template from your DocuSign account in a new signature request envelope.

1. Create a Template

Please log into your DocuSign Demo (Sandbox) account and create a template as follows:

  • Name the template "NDA 1"
  • Download and then use the NDA Sample document for the template
  • Create two roles in the template, "signer 1" and "cc". The role names must be exactly as shown since the software must reference the same role names or the integration will fail.
  • Tag the last page of the template's document: create tabs (fields) for the signer 1 role:
    • A Signature tab
    • A Name tab: the full name of the signer
    • A Date Signed tab
    • A Text tab: the name of the company
    You can also use the Advanced properties of the Text tag to change the tag's Data Label to "company". The last page of the document should now look similar to the following:

2. Envelope Data

Complete this form or use the defaults.
